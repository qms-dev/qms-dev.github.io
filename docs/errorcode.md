## 统一错误码

| Code | 命名        | 描述                                                 |
| :--- | :---------- | :--------------------------------------------------- |
| 0    | **OK**      | 没有错误                                             |
| -1   | **Error**   | 发生错误，检查错误消息和错误详细信息以获取更多信息。 |
| -2   | **Pending** | 表示过程正在处理当中，请稍后重新请求                 |
